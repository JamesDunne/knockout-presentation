KnockoutJS

A ViewModel data binding library

What is Knockout?

“Knockout is a JavaScript library that helps you to create rich, responsive display and editor user interfaces with a clean underlying data model. Any time you have sections of UI that update dynamically [...], KO can help you implement it more simply and maintainably.”

Why Knockout?

  • Declarative bindings
  • Automatic UI refresh
  • Dependency tracking
  • Templating
  • Very low learning curve

Declarative bindings: Easily associate DOM elements with model data using a concise, readable syntax

Automatic UI Refresh: When your data model's state changes, your UI updates automatically

Dependency Tracking: Implicitly set up chains of relationships between model data, to transform and combine it

Templating: Quickly generate sophisticated, nested UIs as a function of your model data

Why Knockout?

  • Separation of HTML presentation from JavaScript logic
  • Ideal for collaborative environment and division of labor
  • View model gets updated automatically when UI changes and vice versa
  • No need for manual jQuery event handling
  • Much less code than jQuery to express complex data bindings

Binding

Binding


<span data-bind="text: message">Default text.</span>

<script>
  function ViewModel() {
    this.message = ko.observable("Hello!");
  }
  ko.applyBindings(new ViewModel());
</script>

function TicketsViewModel() {
  this.tickets = [
    { name: "Economy", price: 199.95 },
    { name: "Business", price: 449.22 },
    { name: "First Class", price: 1199.99 }
  ];
  this.chosenTicket = ko.observable();
  this.resetTicket = function() { this.chosenTicket(null); };
}
ko.applyBindings(new TicketsViewModel());

Observables

  • ko.observable()
    creates an observable value.
  • Subscribe to it to receive notifications when its value changes.
  • Update its value to have subscribers notified.

Computed Observables

  • ko.computed(function() {})
    creates a computed observable.
  • Automatically register subscriptions during evaluation.
  • Re-evaluated when subscriptions change.

function ViewModel() {
  this.a = ko.observable(1);

  // Computed observable `b` subscribes to `a`, gets its value
  // as a number and adds 1.
  this.b = ko.computed(function() {
    return Number(this.a()) + 1;
  }, this);
}
